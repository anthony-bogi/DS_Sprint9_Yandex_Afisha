# Sprint 9. Анализ лояльности пользователей Яндекс Афиши

## Описание проекта

Цель проекта - исследовать факторы, влияющие на возврат пользователей на платформу Яндекс Афиша

В рамках проекта выполнены:

- выгрузка данных из PostgreSQL;
- предобработка данных;
- приведение выручки к единой валюте (рубли);
- анализ пользовательских сегментов;
- исследование факторов удержания пользователей;
- корреляционный анализ признаков профиля пользователя.

## Установка проекта

### 1. Клонировать репозиторий

```bash
git clone <ссылка-на-репозиторий>
cd <папка-проекта>
```

### 2. Создать виртуальное окружение

Windows:

```bash
python3 -m venv venv
source venv\Scripts\activate
```

Linux/macOS:

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

---

## Настройка подключения к базе данных

Создайте файл `.env` в корневой папке проекта.

Пример содержимого:

```env
DB_USER=your_user
DB_PASSWORD=your_password
DB_HOST=your_host
DB_PORT=6432
DB_NAME=your_database
```

В репозиторий файл `.env` не загружается.

Для примера структуры используется файл:

```text
.env.example
```
